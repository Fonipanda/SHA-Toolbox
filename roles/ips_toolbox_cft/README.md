Operation Parameters
====================
check
-----

Check installation, partners and version of CFT

=> No additional parameter

list
----

List partners

=> No additional parameter

start(linux/AIX)
-----

Start process CFT

=> No additional parameter

stop(linux/AIX)
-----

Stop process CFT

=> No additional parameter

status
-----

Check status of CFT process

=> No additional parameter

Dependencies
============

None

Example
=======

```
#start CFT process  
- hosts: servername
  tasks:
    - include_role:
        name: ips_toolbox_cft
      vars:
        cft_operation: "start"
        
```
